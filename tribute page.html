Introduction

JavaScript is a versatile language used for web development.

It runs in browsers and on servers (Node.js).

It supports functional, object-oriented, and event-driven styles.

You can embed it in HTML using the <script> tag.

JavaScript syntax is C-style.

Variables, functions, and control flow are core parts.

It interacts with the DOM for UI manipulation.

Modern JS supports classes and modules.

Browser APIs like fetch() allow AJAX calls.

Use developer tools to debug in real time.

console.log('Hello, JS');
Variables

Use let, const, or var.

let allows reassignment; const does not.

var is function-scoped and outdated.

  • let age = 25;
  • const name = "Alice";
  • var score = 100;

Avoid using var in modern code.

Always declare variables before use.

let message = "Hi";
Functions

Functions are declared using function or as arrow functions.

They improve reusability and structure.

function greet(name) { return "Hello " + name; } const add = (a, b) => a + b;

Arrow functions are concise.

You can pass functions as arguments (higher-order functions).

  • Pure functions
  • Callback functions
  • Recursive functions

Use parameters to make functions dynamic.

Loops

Loops repeat blocks of code.

Types: for, while, do...while, forEach

Use break to exit early.

for (let i = 0; i < 5; i++) { console.log(i); } let i = 0; while (i < 5) { i++; }

Use loops to process arrays.

  • for loops
  • while loops
  • for-of and for-in

Array methods can replace many loops.

Conclusion

JavaScript is essential for interactive web pages.

It has evolved with ES6 and beyond.

Modern JS supports OOP, async/await, and modules.

Used in both frontend and backend (Node.js).

export default function App() {}
  • HTML + CSS + JS = Web
  • Use frameworks like React
